**NextGen Expense Tracker** is a personal project built with **Python and Streamlit** to practice building interactive dashboards. Track your income and expenses, visualize analytics, and manage your budget effectively in a user-friendly interface.

---

## Features

- **Income & Expense Tracking:** Easily add and manage transactions  
- **Recurring Transactions:** Automate monthly bills or income  
- **Analytics & Charts:**  
  - Income vs Expense bar chart  
  - Expense by Category pie chart  
  - Monthly trends  
- **Budget Alerts:** Notifications when spending exceeds limits  
- **Export Reports:** Download transactions as CSV or Excel  

---

## Tech Stack

- **Backend:** Python 3.10+ with Streamlit  
- **Database:** SQLite via SQLAlchemy  
- **Data Analytics & Visualization:** Pandas, Matplotlib, Seaborn  

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/NextGen-Expense-Tracker.git
cd NextGen-Expense-Tracker
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the Application:

```bash
streamlit run main.py
```

4. Open the app in your browser at:

```bash
http://localhost:8501
```

---

# Usage

- Create an account or login
- Add income and expense transactions
- View analytics charts and monthly trends
- Set monthly budget and track spending alerts
- Export your transaction reports as needed

---

# Purpose
This project was created for learning and practice, focusing on Python, Streamlit, databases, and data visualization. It demonstrates a modular and professional approach to building a full-featured personal finance tracker.
